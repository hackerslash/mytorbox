MyTorbox

Connect your TorBox account to generate a personal Stremio install link. Nothing is stored on this server — your keys are encoded into the addon URL itself.

Install in Stremio

Nuvio has no direct install link — tap "Install in Nuvio" to copy the URL, then in the app go to Settings → Addons → Add Addon and paste it.

Anyone with this URL can access your TorBox library — treat it like a password.

Add your own IMDb id + stream URL — it appears as a separate "Custom Streams" catalogue in Stremio and deletes itself 3 hours later.
